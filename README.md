# Simple Student Registration Portal

A beginner-friendly ASP.NET Core web application built using Razor Pages. This project demonstrates the basics of connecting a web interface (HTML) to a server-side backend (C#). 

Users can enter their name into a form, submit it, and instantly receive a personalized registration confirmation message.

## Features
* **Simple Form Binding:** Connects HTML input fields directly to C# variables.
* **Server-Side Processing:** Uses `OnPost()` to handle form submissions securely.
* **Dynamic UI:** Displays a success message without needing complex JavaScript.

## Technologies Used
* C#
* ASP.NET Core (Razor Pages)
* HTML

1. Ensure you have the [.NET SDK](https://dotnet.microsoft.com/download) installed on your computer.
2. Clone this repository or download the project files.
3. Open the project folder in Visual Studio Code.
4. Open the terminal (`Terminal` -> `New Terminal`).
